CMS migration project

I was hired on a temporary contract to help the Council of Europe on a huge CMS migration project. They had an old legacy system they needed to migrate to a new standard EMC CMS system.

I was in close contact with all the key players in the project, incl. several vendors and internal clients using the CMS, trying to help the CoE design/architect a solution which best fulfilled the requirements of the users and maintained most of the functionality of the previous CMS system.

  • Date: July 2014
  • Client: Council of Europe
  • Categories: Enterprise CMS, Architecture, Migration

Danrent.dk

Bootstrapped web startup in Barcelona

Danrent.dk was developed in Rails with a REST API and Mongo DB datastore. I started the project with my friend and partner Alex from Catalonia, and we struggled for almost a year trying to launch the business with no startup capital. In the end we had too few resources and we disagreed about the frontend/layout of the application and strategies, so went separate ways.

I finished the backend solution, but didn't find the front end frameworks such as Ember or Angular maure enough at the time. I might want to continue this project in the future and add a nice frontend to complete the project.

  • Period: 2012-2013
  • Client: Danrent.dk (closed)
  • Categories: Lean Startup, Rails, Mongo DB, Bootstrap

SOA IT Architect

Architecting for the Enterprise

I was head-hunted to be chief architect for the EFI (Et Fælles Indrivelsessystem - one common collection system) the key system for SKAT which was to handle all collection of all payments owed to the Danish Tax agency, with over 800 client systems to be integrated via SOA.

I was in close contact and collaborated with various chief architects from our main vendors, CSC, KMD and IBM on this project and other similar projects all to be hosted on the same SOA infrastructure/platform under construction, based on BEA products.

  • Period: 2005-2006
  • Clients: Handicap styrelsen, Arbejdsmarkedsstyrelsen, Energistyrelsen, Naturgas Fyn
  • Categories: SOA, middleware, enterprise architecture, requirement specs

After finishing my Computer Science master degree at Copenhagen University I was hired by Devoteam as SOA Architect.
